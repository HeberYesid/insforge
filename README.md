# Insforge

Proyecto de prueba para la SDK de Insforge — app Next.js con autenticación y base de datos.

## Tecnologías

- **Framework:** Next.js 15, React 19
- **SDK:** @insforge/sdk
- **Estilos:** Tailwind CSS
- **Base de datos:** Drizzle ORM (migraciones incluidas)

## Estructura

```
Insforge-test/
├── src/              # Código fuente
├── migrations/       # Migraciones de base de datos
├── public/           # Assets estáticos
└── .env.example      # Variables de entorno de ejemplo
```

## Instalación

```bash
cd Insforge-test
cp .env.example .env   # Configurar variables de entorno
npm install
npm run dev
```
